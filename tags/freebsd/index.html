Fri, Jan 13, 2023 at 11:16
backups / duplicacy / freebsd / homelab / restic / truenas

Restic Backups on TrueNAS I’ve been backing my homelab up with duplicacy (See Backing Up the Clusting Using Duplicacy), but I’m fed up with it returning a 0 exit code even if there’s a problem with the backup. This makes me have to do a lot of annoying rummaging through log output to be sure that a backup actually worked, so I decided to switch to restic. In this blog entry, I’m going to explain how to create a jail in TrueNAS, mount directories you want to back up into the jail, install restic, and how to use it to back up to Backblaze b2.
